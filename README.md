# Поймай своё время

Мобильный лендинг с браузерной мини-игрой для студии красоты «Твоё время».

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
npx serve .
```

или

```bash
python -m http.server 8080
```

Затем откройте в браузере `http://localhost:8080` и включите режим мобильного устройства (DevTools → Toggle device toolbar).

## Структура

```
index.html      — экраны лендинга
css/style.css   — стили и брендовая палитра
js/game.js      — игровая механика и логика результата
js/app.js       — навигация, копирование фразы, toast
assets/logo.png — официальный логотип (глубокий изумруд #08281F, обрезанный PNG)
assets/logo-source/ — исходники всех цветовых вариантов из архива студии
```

## Логотип

На лендинге используется вариант **глубокий изумруд (#08281F)** — он лучше всего читается на светлом молочно-кремовом фоне. Файл: `assets/logo.png`.

Другие цветовые варианты лежат в `assets/logo-source/`.
